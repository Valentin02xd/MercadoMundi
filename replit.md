# Mercado Mundi - E-Commerce Website

## Overview
Mercado Mundi is a Spanish-language e-commerce website for PC components and peripherals. It's a single-page application built with vanilla HTML, CSS, and JavaScript.

## Project Structure
- `Indice.html` - Main HTML file (entry point)
- `style.css` - Styling for the entire site
- `script.js` - Client-side JavaScript for product display, cart functionality, and navigation
- `imagenes/` - Product images directory
- `README.md` - Simple navigation reference

## Architecture
- **Frontend**: Pure HTML/CSS/JavaScript (no framework)
- **Storage**: Browser localStorage for cart data
- **Server**: Simple static file server (Python http.server)
- **Port**: 5000 (for Replit webview)
